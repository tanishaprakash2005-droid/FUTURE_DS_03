# Marketing Funnel & Conversion Analysis (Power BI)

## Project Overview
This project analyzes marketing funnel data to understand how users move from leads to converted customers.

The dashboard helps identify:
- Conversion rate
- Lead sources
- Drop-off points in the marketing funnel
- Customer acquisition channels

## Dashboard KPIs
• Total Leads  
• Converted Customers  
• Conversion Rate  

## Visualizations
• Leads by Job  
• Leads by Month  
• Conversion by Channel  
• Marketing Conversion Funnel  

## Tools Used
Power BI  
Data Visualization  
Marketing Funnel Analysis

## Insights
- Conversion rate is around 47%
- Most leads come from cellular contact channel
- Certain job segments convert more customers
